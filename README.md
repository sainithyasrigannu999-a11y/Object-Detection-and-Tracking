# Object-Detection-and-Tracking
> The Object Detection and Tracking System is a real-time computer vision project developed using Python, OpenCV, and the YOLOv8 deep learning model. The main objective of this project is to detect and identify multiple objects from a live webcam feed and display them with bounding boxes and labels in real time. This project demonstrates the practical implementation of artificial intelligence and image processing techniques for smart surveillance and monitoring applications.

> In this project, the webcam is used as the input source to continuously capture video frames. Each frame is processed using the YOLO (You Only Look Once) object detection algorithm, which is one of the fastest and most accurate real-time object detection models. YOLO analyzes the video frame and identifies various objects such as persons, bottles, laptops, mobile phones, chairs, cups, books, and many other commonly used objects. The detected objects are highlighted using rectangular bounding boxes along with their class labels and confidence scores.

> The project uses the OpenCV library for handling video capture, image processing, and displaying the output window. OpenCV helps in reading the webcam frames, drawing bounding boxes, adding text labels, and showing the processed video stream on the screen. The YOLOv8 model from the Ultralytics library is responsible for performing deep learning-based object detection with high speed and efficiency.

> During execution, the system continuously reads frames from the webcam and sends them to the YOLO model for prediction. The model returns the coordinates of detected objects along with confidence values and class IDs. Based on these results, the program draws green-colored rectangles around the objects and displays their names above the boxes. The confidence threshold is adjusted to improve detection accuracy and identify multiple objects effectively.

> An additional feature included in this project is output image saving. The detected frame is automatically saved as an image file named output.jpg, which can later be uploaded to GitHub, included in project reports, or used for documentation purposes. This feature helps in storing detection results for future reference and project demonstrations.

> The project provides hands-on experience with machine learning, deep learning, and computer vision concepts. It also helps in understanding how AI models can process visual data and make intelligent decisions in real time. Overall, the Object Detection and Tracking System is an efficient and practical AI-based application that demonstrates the power of modern computer vision technologies.
